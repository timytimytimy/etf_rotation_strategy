## 🔧 四大核心优化

### 1. 凯利公式仓位管理

**原理**：根据各ETF历史胜率和盈亏比，动态计算最优仓位

**公式**：
```
f = (p × b - q) / b

其中：
f = 最优仓位
p = 胜率
q = 败率 = 1 - p
b = 盈亏比（平均盈利/平均亏损）

实际使用：半凯利（fraction=0.5）降低风险
```

**代码实现**：
```python
def calculate_kelly(win_rate, win_loss_ratio, fraction=0.5):
    """凯利公式计算最优仓位"""
    loss_rate = 1 - win_rate
    kelly = (win_rate * win_loss_ratio - loss_rate) / win_loss_ratio
    kelly = max(0, min(kelly, 1))
    return kelly * fraction  # 半凯利
```

**效果**：
- 高胜率高盈亏比的ETF获得更大仓位
- 自动平衡风险和收益
- 避免过度集中

---

### 2. ATR动态止损

**原理**：根据市场波动率动态调整止损位，避免被正常波动震出

**计算**：
```python
def calculate_atr(high, low, close, window=14):
    """计算ATR（Average True Range）"""
    tr1 = high - low
    tr2 = abs(high - close.shift(1))
    tr3 = abs(low - close.shift(1))
    tr = pd.concat([tr1, tr2, tr3], axis=1).max(axis=1)
    atr = tr.rolling(window=window).mean()
    return atr

# 止损条件
if current_price < buy_price - 2 * atr:
    trigger_stop_loss()
```

**效果**：
- 波动大的ETF止损放宽，避免被震出
- 波动小的ETF止损收紧，及时止损
- 每周一重置止损名单

---

### 3. 多因子排序

**原理**：综合动量、波动率、夏普比率三个因子评分，选出最优ETF

**因子权重**：

| 因子 | 权重 | 说明 |
|------|------|------|
| **动量** | 40% | 25日涨跌幅，越高越好 |
| **波动率** | 30% | 低波动加分，降低风险 |
| **夏普比率** | 30% | 风险调整后收益，越高越好 |

**代码实现**：
```python
# 综合得分
scores[code] = 0.4 * momentum - 0.3 * volatility * 10 + 0.3 * sharpe * 10
```

**效果**：
- 不仅看涨幅，还考虑波动和风险
- 选出的ETF质量更高
- 避免"高波动陷阱"

---

### 4. 相关性过滤

**原理**：选择相关性低的ETF，真正分散风险

**逻辑**：
```python
# 检查与已选ETF的相关性
for candidate in sorted_codes:
    is_low_corr = True
    for selected_code in selected:
        corr = price_history[candidate].corr(price_history[selected_code])
        if abs(corr) > 0.7:  # 相关性阈值
            is_low_corr = False
            break
    
    if is_low_corr:
        selected.append(candidate)
```

**效果**：
- 避免持有高度相关ETF（如多只自由现金流ETF）
- 真正分散风险
- 降低组合波动
