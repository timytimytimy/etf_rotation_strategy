## 三、信号说明

### 3.1 择时信号

| 颜色 | 含义 | 操作 |
|------|------|------|
| 🟢 绿色 | 多头趋势 | 正常持有ETF |
| 🟡 黄色 | 跌破20日均线 | 减仓50% |
| 🔴 红色 | 跌破60日均线 | 清仓，持有货币基金 |

### 3.2 得分计算

```
得分 = 10000 × 斜率 × R²

其中：
- 斜率 = 收盘价序列的线性回归斜率
- R² = 线性拟合的决定系数（0~1）
- 每天对最近25个交易日计算
```

**得分越高 = 趋势越强**

- **时间**：每天14:30自动推送
- **内容**：择时信号 + 持仓建议 + 得分排名

### 4.2 操作步骤

1. **14:50** 打开券商APP
2. 查看今日信号建议的持仓
3. 如果和昨天一样 → 不操作
4. 如果变了 → 卖旧买新
5. 操作完成，第二天重复

### 4.3 注意事项

- 必须在**15:00前**完成操作
- 如果错过，等第二天信号
- 不要随意偏离信号
